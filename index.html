Важно

Если вы не принимаете условия данного Пользовательского соглашения на "Windows Tool", вы не можете использовать это программное обеспечение.

Гарантия

Windows Tool - представляется на условии "как есть". Автор не гарантирует бесперебойную работу программы.

Отказ от ответственности

Ни при каких обстоятельствах Автор не несёт перед Пользователем никакой ответственности за прямой, косвенный, побочный, специальный ущерб.

Политика конфиденциальности

Windows Tool - не собирает и не передает личные данные Пользователя.

Установка нужных компонентов

Позволяет установить или удалить одним кликом в фоновом режиме.

Горячие клавиши

КлавишиДействие
CTRL+SHIFT+F1Игровой режим вкл
CTRL+SHIFT+F2Игровой режим выкл
CTRL+SHIFT+F4Закрывает зависшие процессы
CTRL+SHIFT+F7Приостановка процесса

Инструмент написанный на C# для удобного скачивания последних версий и с возможностью по порядку устанавливать программы.
